## 1.什么是提示词

提示词（Prompt）是指在与 AI 大模型交互时，给大模型发送的命令。可以是一句话、一个问题或一段话。大模型接收到提示词后，会根据提示词生成对应的内容并返回。

### 1.1 糟糕的提示词

- "给我讲一个故事"
- "用 Python 写一个游戏"
- "法国大革命爆发的原因"

这些提示词的问题：没有指定输出范围、没有说明期望的结果格式。

### 1.2 好的提示词特征

- 能够引发大模型的思考
- 明确核心任务
- 约束输出范围
- 清晰表达期望的结果和格式

## 2.提示词工程

提示词工程（Prompt Engineering）是指通过有技巧的编写提示词，使大模型生成的内容尽可能符合我们的预期。这不是一个单纯的技术，而是持续优化提示词的过程。

## 3.编写提示词的六个技巧

### 3.1 定义角色和能力

给 AI 定义一个角色，并指定其能力。

**示例：**
```
你是一名经验丰富的高中历史老师，擅长生动有趣的讲解复杂的历史事件。
```

### 3.2 明确核心任务

清楚地说明要完成什么任务。

**示例：**
```
现在请完成以下任务：
向一名高中生解释法国大革命爆发的主要原因。
```

### 3.3 拆解复杂任务

将复杂任务拆解为多个步骤，告诉大模型怎么思考、怎么执行。

**示例：**
```
请先概述这个事件发生的背景，然后分别从政治、经济、思想三个方面来阐述，每点要配一个例子。
```

### 3.4 指定风格和语气

说明期望的输出风格和语气。

**示例：**
```
请使用简洁口语化、充满热情的语气来回答，避免使用过于学术的术语。
```

### 3.5 指定输出格式

明确要求按照什么格式输出。

**示例：**
```
请严格按照如下格式输出：
1. 历史现场的氛围
2. 危机根源的解析
   - 政治层面
   - 经济层面
   - 思想层面
3. 记忆法宝
```

### 3.6 提供输入输出示例

给出参考示例。

**示例：**
```
在输出的时候，请参考《人类群星闪耀时》这本书当中的叙事风格，不要列出冗长的日期列表。
```

::: tip 说明
- 提示词编写没有固定的顺序，可以根据需要调整
- 不是所有提示词都必须涵盖这六点，根据业务需求适当调整即可
:::

## 4.提示词示例

```
你是一名经验丰富的历史老师，擅长用生动有趣的方式讲述复杂的历史事件。

现在请完成以下任务：
向一名高中生解释法国大革命爆发的原因。

表达方式：
- 语气风格：请使用简洁口语化、充满热情的语气，避免学术黑话
- 内容禁忌：不要使用过于专业的术语

内容与结构要求：
- 开头：请先用一段话生动地描述革命前法国社会的总体氛围和紧张感
- 主体：分别从政治、经济、思想三个维度来阐述原因，每点要配一个例子
- 结尾：提供一个帮助学生记忆的妙招

输出格式：
请严格完整地按照以下框架组织你的回答：
1. 历史现场的氛围
2. 危机根源的解析
   - 政治层面
   - 经济层面
   - 思想层面
3. 记忆法宝
```
