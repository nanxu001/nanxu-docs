## 1.if 条件判断

场景：只有满足指定条件，才会执行对应的代码逻辑。

### 1.1 基本格式

```python
if 要判断的条件:
    条件成立时，要执行对应的操作
```

示例：高考分数超过 680 分就去清华。

```python title = "05.if条件判断.py"
score = 700
if score > 680:
    print("欢迎你来清华读书")
    print("也恭喜即将踏入精彩的大学生活")
print("--------------------")
```

运行结果:

```
欢迎你来清华读书
也恭喜即将踏入精彩的大学生活
--------------------
```

::: info 说明
- 判断条件的结果必须是布尔类型（`True` / `False`）
- 条件后面必须有冒号 `:`
- 归属于 `if` 的代码块需要缩进（建议4个空格），Python 通过缩进来描述代码的层级关系（归属）
:::

### 1.2 if...else...

如果条件成立执行操作1，否则执行操作2。

```python
if 要判断的条件:
    条件成立时，执行对应的操作1
else:
    条件不成立时，执行的操作2
```

```python title = "05.if条件判断.py"
# 闰年判断：非整百年能被4整除，整百年能被400整除
year = int(input("请输入年份："))
if (year % 400 == 0) or (year % 4 == 0 and year % 100 != 0):
    print(f"{year} 是闰年")
else:
    print(f"{year} 是平年")
```

::: info 说明
`else` 不需要条件判断，当 `if` 条件不成立时自动执行。
:::

### 1.3 if...elif...else

多个条件依次判断，满足任意一个即执行对应操作，不再判断后续条件。

```python
if 条件1:
    操作1
elif 条件2:
    操作2
else:
    操作3
```

```python title = "05.if条件判断.py"
# 判断数字是正数、负数还是零
num = int(input("请输入数字："))
if num > 0:
    print(f"{num} 是正数")
elif num < 0:
    print(f"{num} 是负数")
else:
    print(f"{num} 是零")
```

::: warning 注意
从上到下依次判断，前面条件成立后，后续条件便不再判断。
:::
